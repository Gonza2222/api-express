# API Express

## Rutas

### Productos
- `GET /productos`
- `GET /productos/:id`
- `POST /productos`
- `PUT /productos/:id`
- `DELETE /productos/:id`

### Usuarios
- `GET /usuarios`
- `GET /usuarios/:id`
- `POST /usuarios`

### Ventas
- `GET /ventas`
- `GET /ventas/:id`
- `POST /ventas`
